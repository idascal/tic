Fișă de lucru: WinForms C# – Buton, Etichetă și Câmp de Text

Obiectiv

Să creezi o aplicație WinForms simplă în C# care să permită utilizatorului să introducă un text și, atunci când apasă un buton, să afișeze textul într‑o etichetă (sau să afișeze un mesaj de eroare, dacă textul este gol).

Cerințe funcționale

  • Fereastra (Form) va conține:
    • O etichetă (Label) — inițial afișează un text de tip „Introduceți textul:”
    • Un câmp de text (TextBox) — pentru ca utilizatorul să introducă un text
    • Un buton (Button) — când este apăsat, va prelua textul din TextBox și va afișa rezultatul într‑o etichetă
  • Când utilizatorul introduce un text și apasă butonul:
    • Eticheta va fi actualizată să arate: „Ai introdus: <textul introdus>”
    • Dacă câmpul de text este gol, la apăsarea butonului va afișa: „Textul este gol!”
  • (Opțional) Poți adăuga validare — de ex. să nu accepte doar spații goale sau valori numerice, după cerință.

Pași de implementare sugerați

  1. Creează un proiect nou Windows Forms App (Framework sau .NET Core) în Visual Studio.
  2. În designer adaugă:
    • Un Label cu text „Introduceți textul:”
    • Un TextBox
    • Un Button cu text „Afișează”
  3. Fă dublu‑clic pe buton pentru a genera metoda de eveniment button1_Click.
  4. În handler, scrie logica:
    • Citește textBox1.Text
    • Dacă e gol => afișează „Textul este gol!”
    • Altfel => afișează „Ai introdus: <textul>”
  5. Testează aplicația: introdu text, apasă butonul, vezi dacă eticheta se schimbă corespunzător; testează și când textul e gol.

Exemplu de cod (C#)

using System;
using System.Windows.Forms;

namespace FisaWinForms
{
    public partial class Form1 : Form
    {
        public Form1()
        {
            InitializeComponent();
        }

        private void button1_Click(object sender, EventArgs e)
        {
            string input = textBox1.Text.Trim();

            if (string.IsNullOrEmpty(input))
            {
                label1.Text = "Textul este gol!";
            }
            else
            {
                label1.Text = $"Ai introdus: {input}";
            }
        }
    }
}
