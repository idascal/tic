 Crearea tabelelor şi inserarea înregistrărilor cu SQL.


SQL vine de la Structured Query Language și este un limbaj de programare standardizat care este utilizat pentru a gestiona baze de date relaționale și pentru a efectua diverse operații asupra datelor. Bazele limbajului de programare SQL s-au pus în anii 1970, fiind creat în urma lucrărilor lui Edgar F. Codd.

    Instructiunile limbajului Structured Query Language pot fi clasificate in 5 categorii:
  • DDL (Data Definition Language): instrucțiuni de definire a datelor (permit descrierea structurii tabelelor):
  • DML (Data Manipulation Language): instrucțiuni de manipulare a datelor (permit modificarea continutului tabelelor):
  • DQL (Data Query Language): instrucțiuni de interogare a datelor:
  • TCL (Transaction Control Language): instrucțiuni de procesare a tranzacțiilor:
  • DCL (Data Control Language): instructiuni pentru controlul datelor (permit definirea, modificarea si retragerea privilegiilor):
